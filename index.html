Process

Process

The design process often starts with an inspiration. That inspiration can come from an image, brand identity, color palette or any other concept that can be presented in a visually interesting way.

In this case it was an image.

From that image I developed a color palette which would serve as the basis for my design. I then created a brand with cohrent identity and a recognizable visual style. The brand features words starting with “ex” wherever applicable, therefore I created a font based logo around it.

After that I designed a front page layout which is easy to navigate, visually represents the brand and can scale to suit any type of content. It also features a call to action for the user.

From that design I iterated a three page layout which adheres to the set style and shares common navigation elements. I have also created a mobile version of the design, including separate navigation for mobile devices.
